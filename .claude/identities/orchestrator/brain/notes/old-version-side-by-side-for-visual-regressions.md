When a human reports "this looks different than before" about a running
visual/UI app but can't pin down what changed, checking out the OLD
commit's code into a scratch location and running it on a separate port
alongside the current version — a real side-by-side — turns a vague
complaint into a precise, screenshot-comparable diagnosis much faster than
guessing from a code diff alone.
